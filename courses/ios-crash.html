What You'll Learn

This one-day course will teach you the basics of native iOS development using Apple's Xcode development environment. By the end of the day, you will have created your own iPhone application.

  • How to use the Xcode IDE and iOS SDK
  • Hello World
  • Objective-C language basics
  • User Interface design with Storyboards
  • Connecting multiple views and using Tab bars
  • Importing external images
  • Introduction to MapKit and CoreLocation frameworks
  • Sandbox time with 1-on-1 support from our instructional team
